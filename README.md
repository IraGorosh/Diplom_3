## Описание
Данный проект покрывает ui тестами сайт https://stellarburgers.nomoreparties.site/

### Maven

`mvn clean test allure:serve`

### Стек технологий, используемых в проекте

- Java 11
- JUnit 4.13.2
- Apache Maven 3.8.6
- RestAssured 5.3.0
- Selenium 4.8.1
- Allure 2.21.0
- WebDriver Manager 5.3.2
